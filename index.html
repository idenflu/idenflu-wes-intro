기존 창고 시스템의 한계

분산된 레거시 시스템은 운영 가시성을 떨어뜨리고, 대응 속도를 느리게 만듭니다.

파편화된 시스템

설비 모니터링, 작업 지시, 알림이 각기 다른 솔루션에 흩어져 통합 가시성이 없습니다.

지연된 의사결정

실시간 데이터 없이 보고서 기반으로 운영하면 장애 대응과 처리량 최적화가 모두 늦어집니다.

보안·감사의 공백

누가, 언제, 무엇을 변경했는지 추적하지 못하면 규정 준수와 사고 조사가 불가능합니다.

하나의 플랫폼, 완전한 제어

프론트엔드 운영 콘솔부터 백엔드 플랫폼까지, Nova가 창고 운영의 모든 계층을 담당합니다.

2.5D 아이소메트릭 기반 창고 레이아웃 시각화

아이소메트릭 투영 방식으로 창고 전체 구역과 설비 위치를 3D에 가까운 입체감으로 표현합니다. 구역별 색상 코딩으로 상태를 즉시 확인할 수 있습니다.

실시간 설비 모니터링

AGV, 컨베이어 등 각 설비의 상태를 50ms 미만 지연으로 확인합니다. 이상 징후를 조기에 감지해 장애를 예방합니다.

작업 지시 / 태스크 관리

WES 태스크 생성, 우선순위 조정, 완료 추적을 단일 화면에서 처리합니다. 실시간 진행률로 처리량을 최적화합니다.

알림·알람 패널

채널별 알림과 임계치 알람을 중앙 패널에서 확인하고 즉시 대응합니다. 심각도별 분류로 중요한 이슈를 놓치지 않습니다.

IAM — 역할 기반 접근 제어

RBAC 기반 사용자·역할·권한 관리와 메뉴 노출 정책으로 보안 접근을 제어합니다. JWT Bearer 인증으로 안전한 API 통신을 보장합니다.

Audit — 변경 증적 추적

추가 전용 감사 로그와 상관관계 ID로 전사 변경 사실을 추적합니다. 규정 준수와 사고 조사를 위한 완전한 감사 이력을 제공합니다.

Notification — 멀티채널 알림

이메일·SMS·푸시·인앱 채널을 단일 계약으로 발송하고 재시도를 자동화합니다. 중복 발송 방지와 전달 이력 추적을 기본 제공합니다.

Realtime — 실시간 데이터 스트림

Snapshot-first Delta push 구조로 대시보드와 운영 화면을 실시간 동기화합니다. 재연결 시에도 최신 상태를 즉시 복원합니다.
